# yacht

トップダウン視点のヨット（帆走）タイムアタックゲームです。風を読み、帆と舵を操って、ステージ上の **5個の島をすべて巡る** までのタイムを競います。

ブラウザだけで動きます。サーバは不要で、`index.html` を開くだけでプレイできます。

## 遊び方

1. ステージ（1〜8）を選びます。各ステージはシードが固定されており、**島の配置と風の変化はいつ遊んでも同じ**です。
2. 帆を風に合わせて調整し、舵で進路を取りながら、すべての島（緑のドット）へ近づきます。
3. 近づいた島には赤い旗が立ちます（訪問済み）。5個すべて訪問するとクリア。タイムが記録され、ステージ毎のベストがブラウザに保存されます。

## 操作

画面はマウス・タッチのどちらでも操作できます。船は常に画面中央に固定され、進路に応じて**背景が回転**します。

| 場所 | 操作 | 効果 |
|------|------|------|
| 上部中央（タイムの下）の **サイドチェンジ**（⇄）ボタン | タップ／クリック | 船の向き（左右）を反転。画面全体が180°回転（アニメーション） |
| **船首側** の半分 | 上下にドラッグ／スワイプ | メインシート：帆の開き角を調整 |
| **船尾側** の半分 | 上下にドラッグ／スワイプ（帆と上下が逆） | 舵を切る |

- 船首側／船尾側の左右は、サイドチェンジで入れ替わります。
- 右上のハンバーガーメニュー（☰）→「ヘルプ」で、各操作範囲を画面上に図示できます（表示のみ）。
- ミニマップは、船首が右なら右下、船首が左なら左下に表示されます。自船と全島（訪問済み／未訪問）の位置を確認できます。

## 帆走のコツ

- **帆（白）**: 風を受ける面が進行方向への推力を生みます。帆を閉じすぎても開きすぎても進みません。風向きに合わせて開き角を調整しましょう。
- **ウィンデックス（赤）**: マストの先で、見かけの風（apparent wind）が吹いてくる向きを示します。これを目安に帆を合わせます。
- **真正面の風**: 風に向かって真っ直ぐは進めません。舵で向きを変え、ジグザグに詰めます。
- **舵（黄）**: 速度が速いほどよく曲がります。止まっていると曲がれません。
